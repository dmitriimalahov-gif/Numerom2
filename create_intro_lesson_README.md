# Создание нового вводного занятия из файлов

Этот скрипт создает новое вводное занятие на основе файлов из папки `файлы для запуска/NumerOM запуск курса/ водное занятие/для сайта вводное занятие/`

## Что делает скрипт:

1. **Читает файлы:**
   - `Урок_Введение_Теория.txt` - теория урока
   - `Урок_Введение_Упражнения.txt` - практические упражнения
   - `Урок_Введение_Тест.txt` - вопросы теста
   - `Урок_Введение_Челлендж.txt` - 7-дневный челлендж
   - `МЕТОДИЧКА1.pdf` - PDF файл методички

2. **Парсит контент:**
   - Извлекает теорию и структурирует её
   - Парсит упражнения с инструкциями
   - Создает вопросы теста из текстового файла
   - Формирует структуру челленджа на 7 дней

3. **Загружает PDF:**
   - Загружает `МЕТОДИЧКА1.pdf` через API
   - Привязывает PDF к уроку

4. **Создает урок:**
   - Создает новый урок через API `/api/admin/lessons/create`
   - Добавляет весь контент (теория, упражнения, тест, челлендж)
   - Привязывает загруженный PDF

## Использование:

1. **Получите токен администратора:**
   ```bash
   # Откройте браузер:
   # 1. Зайдите на http://localhost:3000/admin
   # 2. Войдите в админ-панель
   # 3. Откройте консоль (F12)
   # 4. Выполните: localStorage.getItem('token')
   # 5. Скопируйте токен
   ```

2. **Запустите скрипт:**
   ```bash
   cd /Users/brandbox/Desktop/numerom/Numerom2
   python3 create_intro_lesson.py <ваш_токен>
   ```

3. **Результат:**
   - Скрипт выведет информацию о созданном уроке
   - Урок будет доступен в админ-панели для редактирования
   - Урок будет виден во вкладке "Обучение" для клиентов

## Структура созданного урока:

- **ID**: `lesson_intro_numbers_<random_id>`
- **Название**: "Вводное занятие: Язык чисел"
- **Модуль**: "Модуль 0: Введение"
- **Теория**: Полный текст из файла `Урок_Введение_Теория.txt`
- **Упражнения**: До 6 упражнений из файла
- **Тест**: До 10 вопросов из файла
- **Челлендж**: 7-дневный челлендж из файла
- **PDF**: Загруженный файл `МЕТОДИЧКА1.pdf`

## После создания:

1. Откройте админ-панель → Управление уроками
2. Найдите созданный урок по названию или ID
3. При необходимости отредактируйте контент
4. Проверьте отображение во вкладке "Обучение"

## Примечания:

- PDF файл загружается автоматически и привязывается к уроку
- Правильные ответы в тесте по умолчанию установлены на вариант B (индекс 1)
- При необходимости правильные ответы можно исправить в админ-панели
- Все файлы должны быть в кодировке UTF-8
