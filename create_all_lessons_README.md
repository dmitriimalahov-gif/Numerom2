# Автоматическое создание всех занятий (0-9)

Этот скрипт автоматически создает все 11 занятий (вводное + 0-9) из папки "файлы для запуска".

## Что делает скрипт:

1. **Обрабатывает все занятия:**
   - Вводное занятие (из папки " водное занятие")
   - Урок 0: Число проблемы
   - Урок 1: СУРЬЯ (Солнце)
   - Урок 2: ЧАНДРА (Луна)
   - Урок 3: ГУРУ (Юпитер)
   - Урок 4: РАХУ
   - Урок 5: БУДДХИ (Меркурий)
   - Урок 6: ШУКРА (Венера)
   - Урок 7: КЕТУ
   - Урок 8: ШАНИ (Сатурн)
   - Урок 9: МАНГАЛ (Марс)

2. **Для каждого занятия:**
   - Читает файлы теории, упражнений, теста и челленджа
   - Загружает все PDF файлы (берет первый как основной)
   - Загружает все Word файлы (берет первый как основной)
   - Парсит контент и создает структуру урока
   - Создает урок через API

3. **Автоматически определяет:**
   - Папку с файлами (ищет подпапку "для сайта" или использует саму папку)
   - Все типы файлов по названиям
   - Структуру контента из текстовых файлов

## Использование:

1. **Получите токен администратора:**
   ```bash
   # Откройте браузер:
   # 1. Зайдите на http://localhost:3000/admin
   # 2. Войдите в админ-панель
   # 3. Откройте консоль (F12)
   # 4. Выполните: localStorage.getItem('token')
   # 5. Скопируйте токен
   ```

2. **Запустите скрипт:**
   ```bash
   cd /Users/brandbox/Desktop/numerom/Numerom2
   python3 create_all_lessons.py <ваш_токен>
   ```

3. **Скрипт автоматически:**
   - Обработает все 11 занятий по порядку
   - Загрузит все медиафайлы (PDF, Word)
   - Создаст уроки с полным контентом
   - Выведет статистику успешных/неуспешных созданий

## Структура файлов в папках:

Скрипт ищет следующие файлы в каждой папке занятия:
- `*Теория*.txt` - теория урока
- `*Упражнения*.txt` - упражнения
- `*Тест*.txt` - вопросы теста
- `*Челлендж*.txt` - челлендж
- `*.pdf` - PDF файлы (первый используется как основной)
- `*.docx`, `*.doc` - Word файлы (первый используется как основной)

## Результат:

После выполнения скрипта:
- Все занятия будут доступны в админ-панели для редактирования
- Все занятия будут видны во вкладке "Обучение" для клиентов
- Все медиафайлы будут загружены и привязаны к урокам

## Примечания:

- Скрипт обрабатывает занятия последовательно
- При ошибке на одном занятии, скрипт продолжит обработку остальных
- Правильные ответы в тестах по умолчанию установлены на вариант B
- При необходимости правильные ответы можно исправить в админ-панели
- Все файлы должны быть в кодировке UTF-8

## Время выполнения:

- Примерное время: 5-15 минут (зависит от размера файлов и скорости интернета)
- Для каждого занятия: загрузка PDF/Word + создание урока
