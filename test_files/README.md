# Тестовые файлы для первого занятия

В этой папке находятся тестовые файлы для проверки загрузки медиафайлов в уроки.

## Файлы

- `test_video.mp4` - тестовый видео файл (минимальный размер)
- `test.pdf` - тестовый PDF документ
- `test.docx` - тестовый Word документ

## Способ 1: Автоматическая загрузка через скрипт

1. Получите токен администратора:
   - Войдите в админ-панель на http://localhost:3000
   - Откройте консоль браузера (F12)
   - Выполните: `localStorage.getItem('token')`
   - Скопируйте токен

2. Запустите скрипт:
```bash
cd test_files
python3 upload_test_files.py <ваш_токен>
```

Скрипт автоматически:
- Загрузит все три файла (видео, PDF, Word)
- Сохранит их в первое занятие (`lesson_numerom_intro`)
- Выведет результаты загрузки

## Способ 2: Ручная загрузка через админ-панель

1. Откройте админ-панель: http://localhost:3000/admin
2. Перейдите в раздел "Управление уроками" или "Редактор уроков"
3. Найдите первое занятие ("Первое занятие NumerOM" или ID: `lesson_numerom_intro`)
4. Нажмите "Редактировать"
5. В секции загрузки медиафайлов (в верхней части):
   - Загрузите `test_video.mp4` в раздел "Видео"
   - Загрузите `test.pdf` в раздел "PDF"
   - Загрузите `test.docx` в раздел "Word"
6. **ВАЖНО**: Нажмите кнопку "Сохранить урок" после загрузки всех файлов
7. Откройте урок во вкладке "Обучение" и проверьте, что все файлы отображаются в шапке урока

## Проверка результата

После загрузки файлов:

1. Войдите в личный кабинет (не админ-панель)
2. Перейдите во вкладку "Обучение"
3. Откройте первое занятие
4. В шапке урока должны отображаться кнопки:
   - "Смотреть видео" (если загружено видео)
   - "Просмотреть PDF" и "Скачать PDF" (если загружен PDF)
   - "Открыть Word" (если загружен Word)

## Важные замечания

- После загрузки файлов **ОБЯЗАТЕЛЬНО** нажмите "Сохранить урок", иначе файлы не будут сохранены в БД
- Файлы загружаются через endpoints для консультаций (`/api/admin/consultations/upload-*`)
- Word файлы загружаются через специальный endpoint для уроков (`/api/admin/lessons/upload-word`)
- Все файлы должны быть доступны для просмотра клиентам через соответствующие endpoints
